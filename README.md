# Logos

Used for generating the logo for RavEngine.

### Building
```sh
git clone https://github.com/RavEngine/Logos --depth=1 --recurse-submodules
mkdir build
cd build
cmake -DCMAKE_BUILD_TYPE=Release ..
cmake --build . --config release --target install
```
